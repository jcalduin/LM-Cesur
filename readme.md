# LM - Ejercicios JavaScript (Manipulacion del DOM)

Repositorio de practicas de JavaScript orientadas al aprendizaje de **manipulacion del DOM** en navegador.

El proyecto esta organizado por ejercicios independientes y progresivos, enfocados en seleccionar elementos, responder a eventos y modificar contenido/estilos en tiempo real.

## Objetivo

- Aprender a interactuar con el DOM usando JavaScript puro.
- Practicar eventos de usuario como `click`.
- Aplicar `classList`, `textContent`, `value` y cambios de atributos (`src`).
- Introducir validaciones basicas de formularios en cliente.

## Contenido del repositorio

- `Ejercicio-01`: cambio de texto en parrafos y alternancia entre texto original/cambiado con botones.
- `Ejercicio-02`: intercambio visual de dos imagenes usando clases CSS.
- `Ejercicio-03`: mostrar/ocultar un parrafo con `classList.toggle()` y cambio del texto del enlace.
- `Ejercicio-04`: cambio de color de un bloque (`div`) al pulsar un boton.
- `Ejercicio-05`: lectura de un `input` y pintado del resultado en pantalla.
- `Ejercicio-06`: comprobacion simple de formulario (campo vacio frente a campo rellenado).
- `Ejercicio-07`: cambio del fondo de la pagina mediante alternancia de clase en `body`.
- `Ejercicio-08.1`: mini galeria de imagenes con imagen principal y miniaturas seleccionables.
- `Ejercicio-08.2`: version mejorada de galeria usando `querySelectorAll` y eventos aplicados en bucle.
- `Ejercicio-09`: carrusel de imagenes con botones `Anterior` y `Siguiente` usando indices circulares.
- `Ejercicio-10`: validacion basica de formulario (nombre, correo y mensaje) con mensajes de error.

## Estructura

Cada carpeta de ejercicio incluye:

- `index.html`
- `script.js`
- `style.css` (excepto en `Ejercicio-01`, que solo usa HTML + JS)

Ademas, el repositorio incluye el documento `Ejercicios manipulacion DOM con JavaScript.pdf` como material de apoyo.

## Enfoque

Este repositorio es **formativo**: no busca ser una aplicacion final, sino una coleccion de practicas para consolidar fundamentos de JavaScript del lado del cliente y manipulacion del DOM.
